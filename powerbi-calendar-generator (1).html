💡 Usage Instructions:
1. Configure your settings (default uses dynamic date range from fact table)
2. Click "Generate DAX Code" to create your calendar table
3. Copy the generated DAX code using the "Copy Code" button
4. In Power BI Desktop, go to "Modeling" → "New Table"
5. Paste the code and press Enter
6. Mark as Date Table: Right-click table → "Mark as date table" → Select [Date] column
7. Create relationships with fact tables using the [Date] column

⚡ Best Practices Implemented:
• Uses MINX/MAXX with ALL() for reliable date ranges
• DateKey as integer (not text) for better performance
• Proper VAR syntax for readability
• No trailing commas (DAX compliant)
• Simplified boolean expressions (no redundant IF statements)
