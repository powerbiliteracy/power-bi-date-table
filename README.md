📅 Power BI Calendar Table Generator

A fully customizable calendar (date dimension) generator for Power BI, designed to help you quickly create production-ready date tables with best-practice DAX, M code, or SQL.

This tool removes the repetitive work of building calendar tables by giving you a flexible, UI-driven way to generate exactly the structure you need for your data model.

🚀 What It Does

Generate a complete, optimized calendar table with:

Dynamic or fixed date ranges
Standard date hierarchy (Year, Quarter, Month, Week, Day)
Business logic flags (Weekday, Weekend, Period Start/End)
Fiscal calendar support
Relative date indicators (Today, Current Month, etc.)
Offset calculations (Days/Months from today)
Holiday tagging by country
Multiple export formats:
DAX (Power BI)
Power Query (M)
SQL
⚡ Why Use This

Building a proper date table is critical in Power BI, but it is often:

Repetitive
Error-prone
Inconsistent across models

This generator ensures:

✅ Consistent modeling standards
✅ Performance-friendly design
✅ Clean, readable code
✅ Faster development
🧠 Best Practices Built In
Uses MINX / MAXX with ALL() for reliable date ranges
Integer-based DateKey for performance
Clean variable structure (VAR) for readability
No redundant logic or unnecessary calculations
Ready for “Mark as Date Table” in Power BI
🎯 Who This Is For
Power BI developers
Data analysts
BI consultants
Anyone building dimensional models
💡 How to Use
Configure your calendar settings
Generate your code
Copy the output
Paste into Power BI (“New Table”)
Mark as Date Table and relate to your fact tables
