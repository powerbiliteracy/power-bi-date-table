<h2>📅 Power BI Calendar Table Generator</h2>

<p>
A fully customizable calendar (date dimension) generator for <strong>Power BI</strong>, 
designed to help you quickly create production-ready date tables with best-practice DAX, M code, or SQL.
</p>

<p>
This tool removes the repetitive work of building calendar tables by giving you a flexible, 
UI-driven way to generate exactly the structure you need for your data model.
</p>

<hr>

<h3>🚀 What It Does</h3>

<ul>
  <li>Dynamic or fixed date ranges</li>
  <li>Standard date hierarchy (Year, Quarter, Month, Week, Day)</li>
  <li>Business logic flags (Weekday, Weekend, Period Start/End)</li>
  <li>Fiscal calendar support</li>
  <li>Relative date indicators (Today, Current Month, etc.)</li>
  <li>Offset calculations (Days/Months from today)</li>
  <li>Holiday tagging by country</li>
  <li>Multiple export formats:
    <ul>
      <li>DAX (Power BI)</li>
      <li>Power Query (M)</li>
      <li>SQL</li>
    </ul>
  </li>
</ul>

<hr>

<h3>⚡ Why Use This</h3>

<p>Building a proper date table is critical in Power BI, but it is often:</p>

<ul>
  <li>Repetitive</li>
  <li>Error-prone</li>
  <li>Inconsistent across models</li>
</ul>

<p>This generator ensures:</p>

<ul>
  <li>✅ Consistent modeling standards</li>
  <li>✅ Performance-friendly design</li>
  <li>✅ Clean, readable code</li>
  <li>✅ Faster development</li>
</ul>

<hr>

<h3>🧠 Best Practices Built In</h3>

<ul>
  <li>Uses <code>MINX</code> / <code>MAXX</code> with <code>ALL()</code> for reliable date ranges</li>
  <li>Integer-based DateKey for performance</li>
  <li>Clean variable structure (<code>VAR</code>) for readability</li>
  <li>No redundant logic or unnecessary calculations</li>
  <li>Ready for “Mark as Date Table” in Power BI</li>
</ul>
